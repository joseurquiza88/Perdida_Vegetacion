# Análisis de pérdida de vegetación – Chaco Seco, Córdoba (2000-2024)

Este repositorio contiene el análisis cuantitativo de la pérdida de vegetación en la porción de la ecorregión del **Chaco Seco** que ocupa la provincia de Córdoba, durante el período **2000-2024**.

## Descripción

- Se calculan pérdidas anuales y acumuladas por departamento.
- Se generan gráficos comparativos y acumulativos de pérdida de vegetación.
- Se incluyen análisis relativos considerando el tamaño de cada departamento dentro del Chaco Seco.

- Los scripts están en Python y R.
